👥 Usuarios del panel

La autenticación se maneja con Supabase Auth. Para crear nuevos administradores o cambiar contraseñas, usá el panel de Supabase → Authentication → Users.

—
Cargando...
💡 Para cambiar tu contraseña, andá al panel de Supabase con tu cuenta o usá la opción "¿Olvidaste tu contraseña?" en la pantalla de login (próximamente).
¿Cómo agregar otro administrador?
  1. Andá a tu proyecto en supabase.com
  2. Authentication → Users → "Add user"
  3. Cargá email y contraseña, marcá "Auto Confirm User"
  4. En SQL Editor: UPDATE profiles SET role='admin' WHERE email='nuevo@email.com';
